PHP - Les instructions conditionnelles



Le fonctionnement des conditions en PHP est proche de celui vu en Javascript.

Note pédagogique : en PHP, une condition évalue une expression qui devient un booléen (true/false). Attention aux conversions implicites : pour éviter les surprises, privilégie === et !== quand tu compares à des valeurs attendues (ex. 0, "", null).

Instruction if

L'instruction if est l'instruction conditionnelle incontournable. Elle exécute une ou plusieurs instructions, suivant une ou plusieurs conditions.

La syntaxe générale est :

if (condition) {
    // Instructions;
} elseif (condition) {
    // Instructions;
} else {
    // Instructions;
}

Bon réflexe : en PHP, elseif est la forme la plus utilisée (équivalente à else if), et elle améliore souvent la lisibilité.

Exemple :

if ($a > $b) {
    echo "$a est plus grand que $b";
}

Peut s'interpréter par : si $a est plus grand que $b, alors exécuter l'instruction ci-dessus. En fait, la condition renvoie une valeur vrai ou faux.

Si plusieurs instructions doivent être exécutées par la suite il faut ouvrir et fermer des accolades pour indiquer le début et la fin des instructions. 

if ($a > $b) {
    echo "$a est plus grand que $b";
    $b = $a;
} else {
    // (exemple volontairement incomplet dans la version initiale)
}

Point technique : le bloc else est optionnel. Mais si tu l'écris, il doit contenir un bloc complet (else { ... }) ou une instruction valide.

Exemple :

if ($a > $b) {
    echo "$a est plus grand que $b";
} else {
    echo "$a est moins grand que $b";
}

Peut s'interpréter par : si $a est plus grand que $b, alors exécuter l'instruction ci-dessous, sinon (si $a n'est pas plus grand que $b) exécuter l'instruction ci-dessous.

Conforme “prod” : quand tu affiches des valeurs dynamiques issues d'un utilisateur (ex. $_GET, formulaire), n'oublie pas l'échappement HTML : htmlspecialchars($valeur, ENT_QUOTES, 'UTF-8').

Opérateur ternaire

Un opérateur ternaire permet de faire la même chose qu'une condition if mais avec une écriture plus condensée.

L'utilisation de cette syntaxe est peu recommandée car sa lisibilité est moins facile que des conditions écrites avec if...else.

La syntaxe d'une condition ternaire est la suivante :

(condition) ? instruction_si_vrai : instruction_si_faux

Par exemple la condition classique :

if ($age >= 18) {
    $reponse = "majeur";
} else {
    $reponse = "mineur";
}

peut être remplacée par :

$reponse = ($age >= 18) ? "majeur" : "mineur";

qui se lit : est-ce que $age >= 18 ? si oui, $reponse = "majeur", sinon $reponse = "mineur".

Soit dans un exemple complet :

$age = 25;

$reponse = ($age >= 18) ? "majeur" : "mineur";

echo "Vous êtes $reponse.";

Dans cet exemple, $reponse vaut "majeur" car $age est supérieur ou égal à 18. L'affichage à l'écran sera donc : Vous êtes majeur. On peut même économiser la variable $reponse et un peu de code en le réécrivant directement dans une instruction echo comme ci-dessous : 

$age = 25;

echo "Vous êtes " . (($age >= 18) ? "majeur" : "mineur") . ".";

Important (précédence) : sans parenthèses, le ternaire peut donner un résultat inattendu quand il est combiné à la concaténation (.). La version ci-dessus est la forme sûre et lisible.

Instruction switch

L'instruction switch est similaire à une série de conditions if sur une même variable, avec l'avantage d'être beaucoup plus claire au niveau du code : 

$a = 2;

switch ($a) {
  case 0:
    echo "$a = 0";
    break;

  case 1:
    echo "$a = 1";
    break;

  case 2:
    echo "$a = 2";
    break;
}

La variable est testée sur les valeurs définies par case X :

Les instructions break peuvent être optionnelles ; mais dans ce cas, même si la condition de case est remplie, les instructions suivantes continueront à être exécutées.

Point technique : le switch en PHP utilise une comparaison “souple” (proche de ==) dans beaucoup de cas, ce qui peut surprendre avec des chaînes et des nombres. Si tu veux une logique plus stricte et moderne (PHP 8+), tu peux présenter match plus tard : c'est une expression, stricte, et sans “fall-through”.

De plus le script peut être complété par une ligne du type :

default:
    echo "$a n'est pas égal à 0, 1 ou 2";

default sera exécuté automatiquement si aucun des case ci-dessus ne correspond.
