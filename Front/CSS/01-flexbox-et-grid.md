# Flexbox et Grid

## Préambule

Pour vous remettre aux sélecteur CSS, je recommande d'utiliser [CSS Diner](https://flukeout.github.io/) en premier lieu.

## Introduction

Ces deux outils sont très pratiques dans le cadre de l'affichage, grid permet de créer une espèce de découpage en tableau, et Flexbox permet de déplacer les éléments dans un boîte.

Ces deux outils CSS sont indépendants mais peuvent être complémentaires.
Il n'est pas rare qu'ils sont utilisés conjointement, notamment en utilisant flexbox à l'intérieur de grid (le cas inverse n'existant pas à ma connaissance).
