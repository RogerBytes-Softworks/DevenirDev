10 Les objets natifs

Les objets sont des briques essentielles de JavaScript : ils regroupent des données (propriétés) et des comportements (méthodes).

Le langage fournit un ensemble d’objets standards, dits objets natifs, qui permettent de manipuler les chaînes de caractères, les nombres, les dates, les tableaux, etc.

L'objet String

L’objet String représente une chaîne de caractères. Dans le JavaScript moderne, on utilise presque toujours des littéraux de chaîne plutôt que le constructeur new String(). 

// (déconseillé) - objet String construit explicitement
const strObj = new String("Bonjour");

// (recommandé) - chaîne primitive
const message = "Bonjour";
// Les primitives de type string bénéficient automatiquement des méthodes de String

La position du premier caractère d’une chaîne est 0. Par exemple, dans "JavaScript" : 'J' est à la position 0, 'a' à la position 1, etc.

Nom Description
length Propriété qui indique la longueur de la chaîne de caractères.
charAt(index) Retourne le caractère à la position index.
indexOf(texte[, fromIndex]) Retourne la position de la première occurrence de texte (ou -1 si non trouvé).
lastIndexOf(texte[, fromIndex]) Identique à indexOf mais en partant de la fin de la chaîne.
includes(texte) Retourne true si la chaîne contient texte, sinon false.
startsWith(texte) Teste si la chaîne commence par texte.
endsWith(texte) Teste si la chaîne se termine par texte.
substring(début, fin) Retourne la partie de la chaîne entre les indices début (inclus) et fin (exclu).
slice(début[, fin]) Semblable à substring, mais accepte aussi des indices négatifs (à partir de la fin).
toLowerCase() Retourne une copie de la chaîne en minuscules.
toUpperCase() Retourne une copie de la chaîne en majuscules.
trim() Supprime les espaces (et retours à la ligne) en début et fin de chaîne.
split(séparateur) Découpe la chaîne en tableau selon le séparateur (par exemple "," ou " ").
replace(recherche, remplacement) Retourne une nouvelle chaîne où la première occurrence de recherche est remplacée par remplacement. Peut utiliser une expression régulière.

Exemple : normaliser un nom d’utilisateur

// On veut :
// - supprimer les espaces superflus
// - ne garder que des minuscules
// - vérifier que le pseudo fait au moins 3 caractères

function normaliserPseudo(pseudoBrut) {
  const nettoyé = pseudoBrut.trim().toLowerCase();
  if (nettoyé.length < 3) {
    return null; // pseudo trop court
  }
  return nettoyé;
}

const entrée = "   Manuel   ";
const pseudo = normaliserPseudo(entrée);

if (pseudo) {
  console.log(`Pseudo enregistré : ${pseudo}`);
} else {
  console.log("Pseudo invalide.");
}

À partir d’ES6, on dispose également des gabarits de chaîne (template strings) avec les backticks `...` et les expressions ${...}, très pratiques pour construire des chaînes lisibles. 

const prénom = "Ella";
const nbMessages = 5;
const phrase = `Bonjour ${prénom}, tu as ${nbMessages} nouveaux messages.`;
console.log(phrase);

L'objet Math et les nombres

L’objet Math fournit des fonctions mathématiques courantes. Il n’est pas destiné à être instancié : on l’utilise directement : 

const maxVal = Math.max(10, 20); // 20
const racine = Math.sqrt(16);         // 4
Méthode Description
Math.abs(x) Valeur absolue de x.
Math.ceil(x) Arrondi à l’entier supérieur ou égal à x.
Math.floor(x) Arrondi à l’entier inférieur ou égal à x.
Math.round(x) Arrondi au plus proche entier.
Math.max(...valeurs) Plus grande des valeurs passées en argument.
Math.min(...valeurs) Plus petite des valeurs passées en argument.
Math.pow(x, y) x à la puissance y.
Math.sqrt(x) Racine carrée de x.
Math.random() Nombre pseudo-aléatoire entre 0 (inclus) et 1 (exclu).

Math propose aussi des fonctions trigonométriques (sin, cos, tan, etc.) et d’autres utilitaires (Math.trunc, Math.sign, …).

Exemple : tirer un entier aléatoire

// Tirer un entier entre min et max inclus
function entierAléatoire(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

const tirage = entierAléatoire(1, 6); // simulacre d'un lancer de dé
console.log(tirage);

Fonctions liées aux nombres

JavaScript fournit aussi des fonctions et méthodes pour convertir et tester des nombres. Les versions modernes sont disponibles sur l’objet Number.

Nom Description
Number.parseInt(texte[, base]) Convertit la chaîne en entier. Par exemple : Number.parseInt("42") donne 42,
Number.parseInt("10", 2) donne 2.
Number.parseFloat(texte) Convertit la chaîne en nombre à virgule (notation point : "3.14").
Number.isNaN(x) Retourne true si x est exactement NaN. Plus fiable que la fonction globale isNaN().
Number.isFinite(x) Retourne true si x est un nombre fini (ni Infinity, ni -Infinity, ni NaN).

Attention : la fonction globale eval() permet d’exécuter du code construit sous forme de chaîne. Elle est considérée comme dangereuse (problèmes de sécurité, de performance, de lisibilité) et doit être évitée dans le code moderne.

Exemple : calculer un total à partir de champs texte 

// Imaginons un formulaire avec des champs texte qui contiennent des prix
const prixBruts = ["12.50", "3.20", "7.30"];

const prixNumériques = prixBruts.map(valeur => Number.parseFloat(valeur));

const total = prixNumériques.reduce(
  (somme, prix) => somme + prix,
  0
);

console.log(`Total = ${total.toFixed(2)} €`);

L'objet Date

L’objet Date représente un instant dans le temps.

On peut le créer de plusieurs façons :

// Date et heure actuelles
const maintenant = new Date();

// À partir de millisecondes écoulées depuis le 1er janvier 1970 (UTC)
const dateDepuisEpoch = new Date(3600 * 24 * 365 * 40);

// À partir de composants (année, mois, jour, ...) - mois entre 0 et 11
const dateAnniversaire = new Date(2025, 2, 15); // 15 mars 2025

// Avec heures, minutes, secondes
const rendezVous = new Date(2025, 2, 15, 9, 30, 0);

// À partir d'une chaîne de caractères (format reconnu par le moteur)
const dateIso = new Date("2025-03-15T09:30:00");
Méthode Description
getFullYear() Année sur 4 chiffres (ex : 2025).
getMonth() Mois entre 0 (janvier) et 11 (décembre).
getDate() Jour du mois entre 1 et 31.
getDay() Jour de la semaine entre 0 et 6 (0 = dimanche, 1 = lundi, etc.).
getHours() Heures entre 0 et 23.
getMinutes() Minutes entre 0 et 59.
getSeconds() Secondes entre 0 et 59.
getTime() Nombre de millisecondes écoulées depuis le 1er janvier 1970 (UTC).
getTimezoneOffset() Différence en minutes entre le fuseau local et l’heure UTC (souvent un multiple de 60).
toLocaleString() Retourne une représentation locale de la date et l’heure (format dépendant de la langue et des options).
Date.now() Méthode statique qui retourne new Date().getTime().

L’objet Date possède également des méthodes setFullYear, setMonth, setDate, etc. pour modifier une date existante.

Exemple : afficher une date lisible et calculer une durée 

const maintenant2 = new Date();

const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
  hour: "2-digit",
  minute: "2-digit"
};

const texteDate = maintenant2.toLocaleString("fr-FR", options);
console.log(`Nous sommes le ${texteDate}`);

// Calculer le nombre de jours entre deux dates
const début = new Date("2025-01-01");
const fin = new Date("2025-12-31");

const diffMs = fin.getTime() - début.getTime();
const unJourMs = 1000 * 60 * 60 * 24;
const nbJours = diffMs / unJourMs;

console.log(`Il y a environ ${nbJours} jours entre ces deux dates.`);

L'objet Array

L’objet Array représente une liste ordonnée d’éléments. On parle souvent de tableau.

En JavaScript moderne, on utilise principalement la syntaxe littérale []. 

// Création de tableaux
const nombres = [10, 5, 4, 20];
const prénoms = ["Alice", "Bob", "Charlie"];

// Tableau vide et ajout d'éléments
const tCarnet = [];
tCarnet.push("Philippe");
tCarnet[1] = "Léa";

// L'indexation commence à 0
console.log(tCarnet[0]); // "Philippe"
Propriété / Méthode Description
length Retourne le nombre d’éléments du tableau (un entier).
push(élément) Ajoute un élément à la fin du tableau.
pop() Supprime et retourne le dernier élément.
unshift(élément) Ajoute un élément au début du tableau.
shift() Supprime et retourne le premier élément.
join([séparateur]) Construit une chaîne à partir des éléments du tableau, séparés par séparateur (par défaut une virgule).
reverse() Inverse l’ordre des éléments (modifie le tableau).
sort([fonctionDeComparaison]) Trie le tableau. Par défaut, tri lexicographique (par ordre alphabétique des chaînes).
slice(début[, fin]) Retourne une copie partielle du tableau du rang début (inclus) à fin (exclu).
forEach(fonction) Exécute une fonction pour chaque élément (itération simple).
map(fonction) Crée un nouveau tableau contenant le résultat de fonction appliquée à chaque élément.
filter(fonction) Crée un nouveau tableau avec les éléments pour lesquels la fonction retourne true.
reduce(fonction, valeurInitiale) Combine les éléments du tableau en une seule valeur (somme, produit, objet accumulé, etc.).

Exemple : filtrer et transformer des données

// Liste de tâches avec une propriété "terminée"
const tâches = [
  { titre: "Préparer le café", terminée: true },
  { titre: "Lire la documentation JS", terminée: false },
  { titre: "Corriger un bug", terminée: true }
];

// Ne garder que les tâches terminées
const terminées = tâches.filter(t => t.terminée);

// Récupérer uniquement les titres
const titresTerminés = terminées.map(t => t.titre);

console.log(titresTerminés.join(", "));

L'objet Image et les images

Dans les navigateurs, une balise <img> dans le DOM est représentée par un objet de type HTMLImageElement. On peut aussi créer des images dynamiquement en JavaScript.

Pré-charger une image

On peut créer une image en mémoire, lui donner une URL, puis l’utiliser plus tard (par exemple pour un effet de survol). 

// Préchargement d'une image
const imageSurvol = new Image();
imageSurvol.src = "images/bouton-survol.png";

imageSurvol.addEventListener("load", () => {
  console.log("Image de survol prête en mémoire.");
});

Exemple : changer une image au survol (version DOM moderne) 

<!-- Dans le HTML -->
<img
  id="btnPlay"
  src="images/bouton-normal.png"
  alt="Bouton Lecture"
/>
// Dans un fichier .js ou un <script> séparé
const btnPlay = document.getElementById("btnPlay");
const imgNormal = "images/bouton-normal.png";
const imgSurvol = "images/bouton-survol.png";

btnPlay.addEventListener("mouseenter", () => {
  btnPlay.src = imgSurvol;
});

btnPlay.addEventListener("mouseleave", () => {
  btnPlay.src = imgNormal;
});

Ne pas confondre l’objet JavaScript Image (instance créée avec new Image()) avec la balise <img> présente dans le HTML. Les deux représentent des images, mais n’ont pas exactement le même rôle.

Exercice

Écrivez un programme qui permet à l’utilisateur de saisir plusieurs valeurs numériques (avec window.prompt). Ces valeurs seront rangées dans un tableau.

  • La saisie s’arrête quand l’utilisateur saisit la valeur 0.
  • À la fin, le programme doit afficher :
    • le nombre de valeurs saisies (hors 0 de fin),
    • la somme de ces valeurs,
    • la moyenne.

Piste de solution (console)

const valeurs = [];

while (true) {
  const saisie = window.prompt("Entrez une valeur numérique (0 pour terminer) :");

  if (saisie === null) {
    // L'utilisateur a cliqué sur Annuler : on arrête aussi
    break;
  }

  const valeur = Number.parseFloat(saisie);

  if (Number.isNaN(valeur)) {
    window.alert("Ce n'est pas un nombre valide.");
    continue;
  }

  if (valeur === 0) {
    break;
  }

  valeurs.push(valeur);
}

if (valeurs.length === 0) {
  console.log("Aucune valeur saisie.");
} else {
  const somme = valeurs.reduce(
    (acc, v) => acc + v,
    0
  );
  const moyenne = somme / valeurs.length;

  console.log(`Nombre de valeurs : ${valeurs.length}`);
  console.log(`Somme : ${somme}`);
  console.log(`Moyenne : ${moyenne}`);
}
